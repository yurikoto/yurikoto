API

注意

调用本站服务或在本站进行投稿，代表您已仔细阅读并同意使用条款。

概况

目前，Yurikoto提供两个API以供调用，即台词与壁纸API。
当前API版本为V1，使用原生PHP编写。当前版本计划两年内长期支持。接口域名：v1.yurikoto.com，全球CDN加速。

调用量限制

对于网站等场景调用本站接口（正确设置HTTP报文头Referer字段，Ajax等调用方式浏览器会自动设置）每个ip每3秒可调用10次；对于浏览器等场景直接访问本站接口（未设置或未正确设置HTTP报文头Referer字段，如window.open()），每个ip每分钟可调用5次。本站检查HTTP报文头主要目的是防止滥用。

注意

本站的所有台词、壁纸资源仅供个人、小型团体的非盈利性软件、站点使用，严禁商用！如果您发现有站点滥用本站服务，请邮件联系liyu@tigerxly.com。

台词API

每次随机获取一条台词，详见这里。

壁纸API

每次随机获取一张壁纸，可指定类型，详见这里。

投稿

由于服务器带宽与性能不足，目前仅开放台词投稿功能。如果需要投稿壁纸或大量投稿台词，请联系liyu@tigerxly.com，详见这里。
