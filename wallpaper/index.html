壁纸接口

请求地址生成

返回编码：       
壁纸类型：       


请求地址

https://v1.yurikoto.com/wallpaper

访问限制

本站所有接口每个ip每分钟100次。

请求参数

参数 可选 说明 默认值
encode 是 返回编码 见下文
type 是 壁纸类型 all

返回编码

参数 说明
默认 直接重定向到图片
text 返回纯文本（只返回链接）
json 返回json

壁纸类型

参数 说明
all 获取随机壁纸
day 获取白天模式壁纸
night 获取夜晚模式壁纸

返回格式

参数 说明
status 状态
id 壁纸编号
link 壁纸链接
type 壁纸类型

状态

参数 状态码 说明
success 200 成功
failed 400 未知错误
failed 403 非法访问
failed 404 接口不存在
failed 429 请求频率过高
failed 500 服务端发生错误
